Looping in Ruby

  • while loops
  • until loops
  • for loops
  • each do
  • break statement

  • while conditional [do] code end (do is optional) also possible to use begin...[code]...end while conditional

  • until conditional [do]...end - executes code until conditional is true, can also use end...until
  • for variable [, variable...] in expression[do] code, executes once for each element in expression. can also use each
  • (0..X)each do |i|
  • break statement breaks an inner loop given a condition

Looping in Javascript

  • while loops
  • for loops

  • while executes code until a condition returns false also possible to use do ...code ...while condition

  • for (statmnt1; stmnt2; stmnt3) { code } statement 1 defines a beginning condition, 2 an end, and 3 an intermediate calculation
