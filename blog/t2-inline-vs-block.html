Inline Display vs. Block

4/1/2015

The display properties of different elements in CSS make each one suitable for a different purpose. For example Div is a block element which means by nature it takes up the entire width of a web page without allowing any other elements on the same line; thus forcing other elements below the Div. While span, on the other hand, is an Inline element which means that it only takes up the space that it needs to print its characters. This allows other elements to be placed "in-line" with the span. Inline-Block is a nice feature that combines the elements of both and creates an display "block" but without forcing it to take up the entire width of the page and allowing other blocks to be placed side-by-side.
