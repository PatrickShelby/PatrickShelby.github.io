Arrays & Hashes

4/5/15

Arrays are especially useful for linear, sequential information where the values of the array do not correspond to other values. While Arrays can map two-dimensional data, Hashes provide a convenient way to enter data as ordered-pairs. The simplicity of arrays make them convenient to access elements of an array. Hashes often require some specific iteration to return desired values.

Despite this hashes would be very useful in recording map coordinates over 2-dimensional grids or even color values for a pixel grid. While Arrays would work well as the data structure for a list of names or numbers.
